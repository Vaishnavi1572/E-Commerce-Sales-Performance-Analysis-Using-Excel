# 🛒 E-Commerce Sales Performance Analysis

---

## 📌 Project Objective
The objective of this project is to analyze e-commerce sales data to understand **sales performance, customer behavior, product trends, and revenue drivers**, and provide meaningful business insights.

---

## 📂 Dataset Description
The dataset contains transactional data including:
- Customer details  
- Product information  
- Store data  
- Sales transactions  

It covers multiple categories, regions, store types, and time periods.

---

## 🧹 Data Cleaning & Transformation
Performed using **Microsoft Excel**:

- Removed duplicate records  
- Handled missing values  
- Created derived columns (Loyalty Level)  
- Structured raw data into clean tables  
- Built datasets: Customer, Product, Sales  

---

## 📊 Analysis & Visualizations

- 📈 Category-wise Revenue Contribution  
- 📉 Monthly Sales Trend (Line Chart)  
- 🏬 Store Type Performance (Online vs Offline)  
- 💰 Price vs Volume Analysis  
- 👥 Loyalty Level vs Sales  
- 🌍 Region-wise Sales Performance  

---

## 🔍 Key Insights

- 🛍️ **Top Categories:** Sports & Electronics generate highest sales  
- 👥 **Customer Behavior:** Bronze customers contribute highest revenue  
- 🌍 **Regional Trends:** North, South & East perform better than West  
- 🏬 **Store Performance:** Online sales outperform offline stores  
- 💰 **Sales Drivers:** High volume drives revenue more than price  
- 📈 **Trend Insight:** Sales show moderate growth with seasonal fluctuations  

---

## 💡 Recommendations

- Increase marketing during peak months (Jul–Sep)  
- Improve performance in the West region  
- Focus on high-volume product categories  
- Strengthen online sales channels  
- Optimize pricing and inventory strategies  

---

## 🛠️ Tools Used

- Microsoft Excel  
- Pivot Tables  
- Charts & Dashboards  

---

## 📈 Project Outcome
This project provides actionable insights to improve sales efficiency, optimize business strategies, and support data-driven decision-making.
